# Sistemas Distribuídos

Repositório com código Python simples e pipeline CI/CD no GitHub Actions.

**Pipeline (3 estágios):**
1. Lint — análise estática (Ruff)
2. Test — testes unitários (pytest)
3. Build — validação

**Repositório:** [GitHub – link do repositório com as Actions configuradas](https://github.com/SEU_USUARIO/Sistemas_Distribuidos)
